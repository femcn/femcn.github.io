Hexo博客添加不蒜子访客统计

发表于 2020-09-07 | 分类于 blog |
字数统计: 667 | 阅读时长 ≈ 2

我这里的hexo版本是4.0.0，有两种方法配置，选一种就行

一、配置文件方法

打开next主题配置文件\themes\next_config.yml，搜索找到busuanzi_count，把enable全部设置为true，header看下面照写就行（注意冒号后面要有空格），你也可以根据需要修改，比如header写访客数，footer写人，就是前后缀的关系而已。

busuanzi_count:

enable: true

site_uv: true
site_uv_header: <i class="fa fa-user"></i>
site_uv_footer:

site_pv: true
site_pv_header: <i class="fa fa-eye"></i>
site_pv_footer:

page_pv: true
page_pv_header: <i class="fa fa-file-o"></i>
page_pv_footer:

1、安装脚本

2、设置标签

要显示站点总访问量，复制以下代码添加到你需要显示的位置。有两种算法可选：

算法a：pv的方式，单个用户连续点击n篇文章，记录n次访问量。

算法b：uv的方式，单个用户连续点击n篇文章，只记录1次访客数。

4、注意，不蒜子域名可能会有更新，如果没效果，是因为那个js链接失效了

打开themes\next\layout_third-party\analytics\busuanzi-counter，更改成以下连接：<script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"> </script>
