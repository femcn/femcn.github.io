hexo博客添加评论功能

发表于 2020-09-08 | 分类于 blog | | 阅读次数:
字数统计: 285 | 阅读时长 ≈ 1

注：我这里的Next主题版本是5.1.4，如果不行可能版本不同，可以去Next官网看教程

评论系统选择:Valine

配置步骤：

1、注册LeanCloud：https://leancloud.cn/dashboard/login.html#/signup，然后在账号设置那里实名认证

2、创建应用

名称自己起，选开发版

点这里的齿轮图标进入设置

点安全中心，服务开关留下数据存储就行，其他可以关掉，下面的web安全域名把你的博客网址粘贴进去

点应用Keys,准备复制id和key

3、修改配置文件

进入主题配置文件\themes\next\_config.yml,ctrl加F找到valine,enable后面改成true,然后粘贴id和key

保存再部署上传，评论设置到这里就完成了

下面是阅读次数功能设置，自带的不蒜子统计也有，选一个来用就行

步骤：

1、回到刚才的leancloud网站，点击存储，创建一个class，名字必须是counter

2、回到主题配置文件，找到leancloud_visitors，同样enable设置成true,粘贴id和key,保存后再部署上传就完成了
